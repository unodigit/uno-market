## ğŸ”§ Plugin Command Sequence

After running the prompt above, the plugin will execute these commands:

### 1. Project Setup
```bash
# Set up the base project structure
/scaffold-component Header --type functional --styling tailwind --tests unit,integration
/scaffold-component ProductCard --type functional --styling tailwind --tests unit
/scaffold-component ProductGrid --type functional --styling tailwind --tests unit,integration
```

### 2. Quality Enforcement
```bash
# Run automated quality checks
/lint-fix-all --fix --type-check --verbose

# Execute comprehensive tests
/run-tests --coverage --reporter=verbose
```

### 3. Design System Configuration
```bash
# Update design tokens with app-specific colors
/design-tokens update --primary-color "#3b82f6" --secondary-color "#10b981"

# Generate Tailwind configuration
/tailwind-config update --validate --backup
```

### Agent Specialization

**UI-Architect** (Claude Sonnet 4.5):
- Designs component architecture and TypeScript interfaces
- Implements React patterns and state management
- Creates responsive layout structures

**Tailwind-Stylist** (GLM 4.6):
- Generates responsive Tailwind CSS utilities
- Implements design system tokens and themes
- Optimizes for mobile-first approach

**Biome-Linter** (GLM 4.6):
- Enforces code quality standards
- Organizes imports and formatting
- Optimizes TypeScript types

**Vitest-Tester** (GLM 4.6):
- Creates comprehensive test suites
- Implements unit and integration tests
- Generates coverage reports

**Monorepo-Orchestrator** (Claude Sonnet 4.5):
- Coordinates parallel agent execution
- Optimizes build performance
- Manages cross-component dependencies
